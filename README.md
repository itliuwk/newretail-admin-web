# newretail-admin-web 
## 演示地址：http://xyz.sxitw.cn/quxia   非真实数据
趣闸后台管理系统的前端项目，基于Vue + Element实现。主要包括设备管理，货道管理，场地管理，库存管理，广告管理，商品管理，权限管理，财务管理，权限管理，分润管理 销售订单 财务等功能。
